Edición con iD en Openstreetmap

#GéneroySoftwareLibre 

El editor iD es el editor de OpenStreetMap por defecto, basado en navegador.

iD es fácil y rápido de utilizar, y permite mapear desde diferentes fuentes de datos como imágenes de satélite y aéreas, GPS, Field Papers o Mapillary.

COMENZANDO CON EL EDITOR ID

  • Abrir el navegador de Internet e ir a la página web de OpenStreetMap en www.openstreetmap.org
  • Iniciar Sesión o crear una nueva cuenta de OpenStreetMap

COMENZANDO CON EL EDITOR ID

  • Ir al área en el mapa que desea editar.

  • Desplazar el mapa manteniendo pulsado el botón izquierdo del ratón y arrastrando el mapa al área deseada o buscar una ciudad o dirección en el buscador

COMENZANDO CON EL EDITOR ID

  • Hacer clic en la flecha pequeña junto a Editar. Luego, hacer clic en Editar con iD (editor en el navegador

Etiquetas en OpenStreetMap

  • Cada etiqueta describe un atributo geográfico de la característica mostrada por ese punto, línea o área.
  • Una etiqueta 'Tag' tiene información considerada Key y value, es decir, una 'Clave' y un 'Valor' 
  • Existen acuerdos sobre el uso de etiquetas que se usan para la mayoría de los propósitos comunes.
  • Rol de la perspectiva de género

Etiquetas en OpenStreetMap

  • Key: Highway

  • Value: Primary

  • Descripción: Carreteras federales libres de dos carriles y atraviesan diversas poblaciones. Vías principales y pueden ser de 2 o 4 carriles
